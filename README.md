# Orbit_Defender
A systems-focused Unity C# prototype demonstrating input handling, state management, and core loop design.

### Key Systems & Concepts
- Input handling (keyboard and mouse-driven UI)
- State management across gameplay and menus
- Deterministic collision and interaction logic
- Progression-driven core gameplay loop
- Dynamic map for easy level navigation
